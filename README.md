# Tetris-C
A Tetris game made in C

![2021-02-17 (2)](https://user-images.githubusercontent.com/46231350/108207183-2934ca00-7106-11eb-9169-a5068fa75408.png)

### How to play

| Controls | Keys |
| ------------- |:-------------:|
| Left      | <kbd>A</kbd> |
| Right    | <kbd>D</kbd> |
| Rotate Clockwise | <kbd>E</kbd> |
| Rotate Counter-Clockwise | <kbd>Q</kbd> |
| Drop    | <kbd>Space</kbd> |
| Pause | <kbd>P</kbd> |
